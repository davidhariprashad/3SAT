How to compile?

No external packages are required.
Successfully compiled on javac 12.0.1
Extract the zipped folder.
Enter within the correct working directory:
javac *.java

How to install?

Other than the JRE, no additional installation is required.

How to run?

Enter exactly one or exactly four arguments.
The first argument is the path to the input file.
The second argument is the path to the output file for the brute force algorithm.
The third argument is the path to the output file for the random algorithm.
The fourth argument is the path to the output file for the genetic algorithm.
For example:
java SAT3 input.txt [arg1 arg2 arg3]
